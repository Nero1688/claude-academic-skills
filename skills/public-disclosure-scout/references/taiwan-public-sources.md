# 台灣官方免費揭露資料源目錄

免費、官方、可研究用的台灣揭露資料源。所有端點與涵蓋範圍會隨時間變動,使用前
以官方站台現況為準;本目錄用於「快速路由與初判」,不保證欄位級精確。

## A. 公開資訊觀測站(MOPS)— 免費源之王

網址:mops.twse.com.tw(新版)/ mopsov.twse.com.tw(舊版端點,部分查詢仍用)。
涵蓋全體上市、上櫃、興櫃、公開發行公司的法定揭露。

| 類別 | 內容 | 學術用途 |
|---|---|---|
| 重大訊息 | 公司即時重大訊息公告 | **事件研究事件源**(治理事件、股利、增減資、契約、裁罰) |
| 財務報表 | 季報、年報、合併報表 | 財務變數(付費庫的免費對照) |
| 年報 | 股東會年報全文(PDF) | text-analytics 的 MD&A/治理段語料 |
| 股東會 | 股東會決議、議事錄、電子投票 | 治理研究(議案、通過率) |
| 董監事 | 持股狀況、設質、改選、酬勞、學經歷 | **家族企業/治理研究關鍵**(親屬關係、酬勞) |
| 內部人 | 持股實際異動、申報轉讓 | 內部人交易研究 |
| 公司治理 | 治理評鑑、獨董、薪酬委員會 | 治理品質變數 |

注意:重大訊息的歷史回補通常逐月查詢;民國日期需轉西元;同日多則需去重歸併。

## B. 證交所/櫃買中心官網(TWSE / TPEx)

- TWSE(twse.com.tw):個股日成交、月營收、三大法人買賣超、融資融券、
  借券、除權息、本益比/殖利率/股價淨值比。
- TPEx(tpex.org.tw):上櫃與興櫃對應資料。
- 學術用途:股價與報酬(事件研究的報酬序列)、法人籌碼、估值比率。
  多為免費 CSV/HTML,量大時注意抓取禮貌與端點限制。

## C. 政府開放資料平台(data.gov.tw)

- 授權明確(政府資料開放授權條款第 1 版),研究用最安心。
- 涵蓋:產業統計、總體經濟、公司登記(經濟部商業司)、勞動、環境、能源等。
- 用途:控制變數、產業層次資料、公司基本背景(統編、設立日、資本額)。

## D. 專責機關揭露

| 機關 | 資料 | 用途 |
|---|---|---|
| TIPO 智慧財產局 | 專利/商標檢索 | 創新研究(專利數、引證) |
| 金管會 | 裁罰案件、行政處分 | 違規/治理失靈事件 |

## E. 免費源 vs 付費庫(TEJ)的分工判斷

| 面向 | 免費官方源 | TEJ(付費) |
|---|---|---|
| 即時性 | 重大訊息最即時 | 有處理延遲 |
| 結構化 | 需自行解析(半結構/PDF) | 已整理成面板 |
| 歷史深度 | 端點限制,回補費工 | 長期序列整齊 |
| 變數加工 | 原始,要自算(如 Tobin's Q) | 現成計算欄位多 |
| 識別碼 | 股票代號/統編 | TEJ 自有碼+對照 |
| 成本 | 免費 | 訂閱 |

決策:**事件日與即時揭露走免費源;整齊的長期面板變數走 TEJ;兩者以統編/代號對接**。
論文方法節要交代兩源涵蓋差異與合併規則,這是審稿人會問的。

## F. 識別碼對接注意

- 上市櫃轉板、公司更名、暫停/恢復交易會造成代號變動;跨期合併用**統一編號**最穩。
- 金融控股與子公司、母子公司合併報表的層級要對齊研究單位。
- 免費源與 TEJ 對接前,先用少量樣本核對代號一致性,再全量合併。
