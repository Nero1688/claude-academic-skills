# 一致性稽核 checklist(管理/財務 + APA/TEJ)

## 1. 假設 ↔ 迴歸表對齊
- [ ] 列出 H1…Hk,各自的應變數與預期方向。
- [ ] 每張迴歸表的「應變數」與其檢定的假設相符(不可把信用評等假設標在 Tobin's Q 表上)。
- [ ] 表註標示的 H 編號正確。
- [ ] 主效果模型 vs 調節/交乘模型分別對應哪些假設,標示一致。
- [ ] 係數方向、顯著性與「假設獲支持/不支持」之結論一致。

## 2. 樣本數一致性
- [ ] 篩選表最終 N = 敘述統計 N = 迴歸觀察值;不一致須說明。
- [ ] 落後一期(t-1)造成的最末/最初年度流失已交代。
- [ ] 敘述統計與迴歸是否同一樣本;若否,註明。

## 3. 篩選聲明 ↔ 敘述統計
- [ ] 「剔除遺漏值」之數字與各變數實際 N 相符。
- [ ] 部分變數(如 TCRI)N 較少時,說明刪除方式(listwise / pairwise)。

## 4. 資料品質界限
- [ ] 不可能值:董事會規模≥1、比率變數落在合理域、虛擬變數∈{0,1}。
- [ ] 離群:max 偏離 mean 是否逾~10 SD;若是,檢查並考慮縮尾。
- [ ] 連續變數縮尾(1%/99%)有無做、有無於文中註明。
- [ ] 【L-002 紅旗】相關係數矩陣中任兩變數 |r|>0.9:疑似共線,或「同一變數誤當兩個放入」(如平方項與原項未置中、虛擬變數代錯)。>0.9 須查是否為誤代或應置中處理,不可放著。
- [ ] 【L-002 紅旗】假說→變數→資料欄位三欄對照:每個假設用到的變數,對到的 TEJ 欄位/代碼是否正確?有無「RPT 誤代 TNFD」這類代碼張冠李戴?

## 4b. 同值重複掃描(L-003 紅旗)
- [ ] 【L-003 紅旗】跨欄格「不同儲存格出現完全相同數值」:t 值、係數、標準誤在不該相同處重複(如兩個不同變數的 t 值都是 38.617),疑似複製貼上貼錯。發現即回溯每個數字的來源輸出檔與位置。
- [ ] 每個進入表格的數字(係數、t、N、R²)可指出來源(哪個 regression output、哪一列)。對不上就停,不自行填補。

## 5. 文字 ↔ 表格
- [ ] 樣本期間、比例(如揭露率)、N、關鍵係數、R²,內文與表格逐一吻合。
- [ ] 交乘項判讀:邊際效果 = β₁ + β₃·M;轉折點 = -β₁/β₃。
- [ ] 二次項判讀:轉折點 = -β₁/(2β₂),並與 U-test、樣本值域對照。

## 6. 引用與表註
- [ ] 內文 APA 引用皆出現在參考文獻清單(無孤兒引用)。
- [ ] 參考文獻清單皆被內文引用(無漏引)。
- [ ] 顯著性星號於表註定義。
- [ ] 表註應變數/標準誤類型(如公司層級叢集)與該表一致。
