# 常用變數種子對照表(2026-07 新增)

商管實證最常用的 20 個變數,Compustat/CRSP 定義 → TEJ 對應的**名稱式**對照。
用途:每次映射從這裡起跑(80 分起點),不必從零推導。

**狀態標記紀律**:
- 【名稱對照】= TEJ 側以官方模組/欄位**名稱**表述,已與 tej-catalog 對得上;
  精確欄位代碼仍須在 TEJ Pro 以智能搜尋核對後才可寫進正式對照表。
- 【推測】= 模組定位有把握、欄位存在性待確認;附驗證方法。
- **本表不含任何臆造的 TEJ 欄位代碼**——代碼欄一律留給核對後的正式產出。

慣例:Compustat 代碼以標準 XPF 名稱(AT、NI、DLTT…);TEJ 財務資料以
「IFRS Finance(國際會計準則)」資料庫為預設來源;金融業另有專表(見注意事項)。

## 注意事項(對照時必檢)

1. **金融業**:第 1、3、5、7 類變數金融業科目結構不同,TEJ 有金融業專表;混合樣本要嘛排除金融業、要嘛分表抓,論文交代清楚。
2. **口徑陷阱**:TEJ 的現成比率欄常有多口徑(稅前/稅後、息前/息後、A/B/C 版),映射時逐字核對文獻公式,不能只看欄位名相似。
3. **頻率**:財務=季/年;股價=日;TESG/TCRI=年。合併時以研究設計的分析單位為準(通常公司-年)。
4. **單位**:TEJ 金額多為千元,比率多為 %;進模型前統一。
5. 逐欄核對完成後,把「TEJ 欄位代碼」欄補進正式三欄對照表——**本種子表永遠不填代碼**,防止未核對的代碼被複製傳播。
