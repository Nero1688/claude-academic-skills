# 出版級圖表版面規範(management-figure)

## 尺寸與解析度
- 單欄圖約 89mm 寬、雙欄約 183mm(對應期刊規範);螢幕預覽用 6.2×4.6 inch 即可。
- 點陣輸出一律 300dpi;投稿同時附 PDF 或 SVG 向量檔。

## 字體與字級
- 無襯線字(Arial/Helvetica 類;本環境用 DejaVu Sans)。
- 軸標 11pt、刻度 9pt、圖例/註記 9pt。圖內不放標題(標題寫在圖說 caption)。

## 線條與框線
- 移除上、右框線(despine);框線 0.8pt。
- 擬合線 2.0–2.2pt;信賴帶半透明填色、無邊線。

## 顏色(色盲友善 Okabe-Ito)
藍 #0072B2、橘紅 #D55E00、綠 #009E73、粉 #CC79A7、黃 #E69F00、淺藍 #56B4E9、灰 #999999。
類別區分除顏色外,另加形狀/線型作第二線索。

## 多面板
- 用 panel label(a, b, c)置於各子圖左上、粗體;子圖間留白充足。

## 管理計量圖種對應假設
- 倒U/U 轉折點圖 → 非線性假設(二次項)。必標轉折點與 β₂ 顯著性。
- 係數森林圖 → 主效果一圖總覽。零線虛線,顯著者上色。
- 交互作用/調節圖 → 調節假設。低/高調節值各一斜率線,看斜率是否翻轉。
- 邊際效果圖 → 調節下 X 對 Y 的邊際效果隨調節值變化(含信賴帶)。
- 分組比較/趨勢圖 → 家族 vs 非家族等次群組描述。
