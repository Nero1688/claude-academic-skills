# 海報版面與印刷規格

## 尺寸速查(pptx page size 直接設為實際尺寸)

| 規格 | 直式 (寬×高 cm) | 橫式 | 備註 |
|---|---|---|---|
| A0 | 84.1 × 118.9 | 118.9 × 84.1 | 國內外研討會最常見 |
| A1 | 59.4 × 84.1 | 84.1 × 59.4 | 系上成果展常用 |

python-pptx 設定:`prs.slide_width = Cm(84.1)`、`prs.slide_height = Cm(118.9)`。
PowerPoint 手動:設計→投影片大小→自訂(上限 142.2cm,A0 沒問題)。

## 傳統三欄式(A0 直式)版面骨架

```
┌──────────────────────────────┐
│ 標題區(高 ~12%):標題85pt+作者/機構/校徽    │
├─────────┬─────────┬──────────┤
│ 欄1       │ 欄2       │ 欄3        │
│ 背景與缺口 │ 研究架構圖  │ 結果圖(主) │
│ 假說       │ 資料與方法  │ 結論與意涵  │
│           │           │ QR+聯絡    │
├─────────┴─────────┴──────────┤
│ 底欄(高 ~5%):參考文獻(精選4-6筆)+致謝    │
└──────────────────────────────┘
```
- 欄間距 ≥2cm;頁邊距 ≥2.5cm(兼作印刷安全區)。
- 閱讀動線:欄內由上而下、欄間由左而右;編號(①②③)輔助引導。

## Better Poster 式(A0 橫式)版面骨架

```
┌────────┬──────────────┬────────┐
│ 側欄L(25%) │   中央區(50%)      │ 側欄R(25%) │
│ 背景/缺口   │                   │ 方法摘要    │
│ 假說一句話  │   主發現一句大字     │ 關鍵數字卡  │
│ 研究架構小圖 │   (60-90pt,置中)  │ 意涵3點    │
│           │   +一張核心圖       │ QR(大)    │
└────────┴──────────────┴────────┘
```
- 中央區背景用主色色塊、白字,對比要在 10 公尺外可讀。
- QR 至少 8×8cm(掃描距離 1m);連結到論文全文或預印本,連結要先測活。

## 字體與色彩

- 字體:中文黑體系(微軟正黑體/思源黑體)、英文 sans-serif(Arial/Calibri);
  海報**不用**襯線內文——遠讀清晰度優先。全張字體 ≤2 家族。
- 色彩:1 主色+1 強調色+灰階;沿用 management-figure 的 Okabe-Ito 色盲友善原則。
  底色建議白/極淺灰(深底全彩印刷大面積易色偏且貴)。
- 行距 1.15–1.3;bullet ≤2 層;段落 ≤4 行。

## 印刷前檢查清單

- [ ] 頁面尺寸=實際輸出尺寸(不是 A4 放大——點陣圖會糊)
- [ ] 圖表全部向量(management-figure 的 PDF/SVG 直插)或 ≥150dpi @實寸
- [ ] 出血:滿版色塊外擴 3–5mm;文字距裁切線 ≥25mm
- [ ] 縮圖 10% 自檢:主發現仍可讀
- [ ] QR 實測可掃(印出 A4 比例測試頁掃一次)
- [ ] RGB→CMYK 色偏抽查:螢光感的藍綠色最容易印壞,送印前問輸出中心
- [ ] 校徽用官方向量版,勿網路抓圖
