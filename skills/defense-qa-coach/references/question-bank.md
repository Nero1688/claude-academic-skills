# 委員提問題庫:六類分類法與追問樹

提問不是隨機的——它長在論文弱點上。掃描論文時按六類定位,每類附「弱點訊號→
必然引來的提問→典型追問樹」。命中越多訊號,該類提問機率越高。

## 第一類:方法論與識別(量化論文的主戰場)

| 弱點訊號 | 必然提問 |
|---|---|
| 自變數是自選擇的(揭露、採用、認證) | 「這是自選擇,你怎麼處理內生性?」 |
| 只有 OLS/FE,無因果識別設計 | 「你敢說這是因果嗎,還是只是相關?」 |
| 樣本期間短或事件少 | 「N 夠嗎?統計檢定力?」 |
| 代理變數與構念距離遠 | 「X 真的量到你說的構念嗎?」 |
| 遺漏變數疑慮(產業、規模效果) | 「你控制了…嗎?沒控會怎樣?」 |

**追問樹範例(自選擇)**:
- Q:「揭露是企業自己選的,內生性怎麼辦?」
  - 你答「加了固定效果」→ 追:「FE 只擋時間不變的遺漏變數,時變的呢?」
    - 你答「加了時變控制」→ 追:「反向因果呢?是揭露帶來績效,還是績效好才敢揭露?」
  - 你答「做了 PSM/Heckman」→ 追:「配對/排除變數的正當性?第一階段工具是什麼?」

## 第二類:理論貢獻(「So what?」戰場)

必問題(每場都有):
1. 「你的理論貢獻**一句話**是什麼?」(答超過三句=沒想清楚)
2. 「這跟 ○○○(該領域經典)的差別在哪?」
3. 「把台灣/你的產業拿掉,這研究還剩什麼?」(戳 context-bound)
4. 「結果不顯著的部分,理論上怎麼交代?」(H 不成立≠沒貢獻,要會轉)

**追問樹(貢獻)**:「貢獻是什麼?」→ 你列三點 → 「三點裡哪一點是**只有你**做得到的?」
→ 你選一點 → 「這一點如果審稿人說『早有人做過』,你怎麼回?」

## 第三類:資料與測量

- 「資料哪來的?TEJ 的這個欄位口徑是什麼?」(答不出口徑=致命)
- 「遺漏值怎麼處理?刪了多少樣本?刪掉的跟留下的有系統性差異嗎?」
- 「Winsorize 在幾 %?為什麼?不縮尾結果變嗎?」
- 「這個變數為什麼用年頻?季頻會不會不一樣?」

## 第四類:穩健性與另類解釋

- 「換一個 Y 的衡量,結果還在嗎?」
- 「你的效果會不會只是 ○○ 的偽裝?(規模/產業景氣/時間趨勢)」
- 「分樣本(金融/非金融、大小公司)結果一致嗎?不一致怎麼解釋?」

## 第五類:實務與政策意涵

- 「主管機關/企業看了你的研究,明天該做什麼不一樣的事?」
- 「效果的經濟顯著性多大?統計顯著但實務上微不足道?」
- (在職生特別注意)「你的實務經驗怎麼看這個結果?」——這是送分題,別答成學術腔。

## 第六類:限制與未來研究

- 「你自己認為最大的限制是什麼?」(考誠實與自知;答「沒什麼限制」=災難)
- 「如果重做一次,你會改哪裡?」
- 陷阱題:「這個限制這麼大,那結論還站得住嗎?」→ 用「界定範圍」防守,不是道歉。

## 黑天鵝題(低機率高殺傷,至少準備三題)

- 委員 A 與委員 B 意見相反,都要你採納 →(應對見 answer-frameworks 第 5 節)
- 「第 X 頁這個數字跟第 Y 頁對不上」→ 口試前先跑 thesis-consistency-audit 就是為了這一刻。
- 「你這個題目五年前就有人做了」→ 準備好與最相近文獻的三點差異表。
